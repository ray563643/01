# 自動排班月曆工具

這是一個純前端的排班工具，可直接用瀏覽器開啟，不需要後端或資料庫。

## 功能

- 月曆式排班畫面
- 正職人員可設定排假天數、指定休、顏色
- 正職在月曆中只顯示休假
- 兼職在月曆中顯示姓名與上班時間
- 店休標紅
- 可設定平日與假日需求人數
- 可設定正職限制：最多連上、每日最少正職、優先排平日休
- 自動排正職休假
- 每日人數不足提醒
- CSV 匯出
- 資料儲存在瀏覽器 localStorage

## Codex cloud 維護方式

這個 repo 是給 Codex cloud 接續維護使用。主入口是 `index.html`，邏輯集中在 `app.js`，樣式在 `styles.css`。

建議下一步：

1. 在 Codex cloud 匯入此 repo。
2. 將靜態版改成 Vite + React 專案。
3. 加入 Supabase 或 Firebase，讓手機與電腦同步同一份排班資料。
4. 整合 PNG/PDF/Excel 匯出。
